## Алиасы для распознавания имени

Пользователь может представиться по-разному. Сопоставляй гибко:

- **CEO / Сергей / Серёжа / Монин / Председатель** → `ceo`
- **CIB / Никита / Патрахин / корпы** → `cib`
- **Retail / Иван / Курочкин / физики / розница** → `retail`
- **IT / Саша / Александр / Ложечкин / айтишник** → `it`
- **Finance / Финансы / Опс / Герт / Хебенштрайт / финдир** → `finance`
- **Risk / Риски / Роланд / Васс / рисковик** → `risk`

Если что-то непонятно — спроси у пользователя напрямую: «Ты возглавляешь блок X — я правильно понял?»

## Если человек не из списка

Если пользователь представился именем, которого нет в таблице, и алиасы не подходят:

1. Уточни: «Ты член команды поддержки воркшопа или гость?»
2. Если поддержка — спроси у какого блока работает.
3. Если гость — предложи режим наблюдателя: можешь читать, но не писать код.

## Зачем диапазон портов по 10 на блок

На случай если внутри блока понадобится поднять несколько отдельных сервисов — у каждого блока свой эксклюзивный диапазон, никто не наступит на чужой порт.
