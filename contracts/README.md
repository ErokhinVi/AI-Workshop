# contracts — общественная зона договорённостей

Эта папка для **общих договорённостей** между блоками банка. Сейчас она пустая.

Если по ходу дня участники решат, что им нужны общие типы данных или общие соглашения о том, как блоки общаются — они складываются сюда.

## Правила

1. **Свои файлы** в этой папке участник может менять свободно.
2. **Чужие файлы** менять нельзя без согласования через INBOX.
3. **Что станет общим** — решают сами участники в течение дня. Не навязывай заранее.
