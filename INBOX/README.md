# INBOX — общий канал между блоками

Это место, где блоки переписываются друг с другом. Если тебе что-то нужно от соседа — твой AI-помощник пишет ему сюда. Если тебе кто-то что-то прислал — ты увидишь это в файле `to_<твой_блок>.md`.

## Как это работает

- Скажи AI-помощнику: «попроси Роланда [что-то]» — он сам напишет нужное сообщение в `to_risk.md`.
- Раз в 10–15 минут AI заглядывает в твой входящий файл и сообщает если есть новые сообщения.
- Никаких ручных действий не требуется.

## Зачем INBOX видим всем

Это **открытый канал** — Виталий и Нерсес во время воркшопа смотрят на эту переписку и обсуждают «как ваше правление договаривается». Один из главных смыслов воркшопа — увидеть, как ваша команда работает с распределённой задачей через AI.

## Формат сообщений (для AI)

```
---
from: <блок_отправитель>
to: <блок_получатель>
date: YYYY-MM-DD HH:MM
status: open | replied | agreed | rejected | done
---

Текст сообщения.
```
